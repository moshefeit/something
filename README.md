## Note for Databricks user: How to gain pull this data to Databricks?

Here is the trick: `wget`

```
%sh

wget http://repleace.this/with-link

```

To see the file you can use `ls -l`

```
%sh
ls -l
```

To see the full path to your file, use `pwd`.

```
%sh
pwd
```
